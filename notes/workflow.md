# Workflow minimo

Uso consigliato di questo repo personale:

1. creare notebook leggero per un filone
2. aggiungere un mini README con stato e caveat
3. usare il notebook come base per una discussion `Analisi`
4. promuovere nel Lab solo se il filone regge davvero

## Regole pratiche

- non copiare qui dati pesanti dal Lab
- non duplicare il contratto tecnico di `dataset-incubator`
- tenere i notebook piccoli e leggibili
- distinguere sempre tra notebook personale e artefatto canonico del Lab
